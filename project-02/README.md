## Mini-Project 2

### Project Overview
This project explores the FIFA 18 Player data with the goal to understand relationships between players rating, potential, and country. In this project there are interactive visualizations, spatial mapping, and regression modeling.

#### Data set
This data set that was used is:
- `fifa18.csv`

#### Spatial data Required:
- Country shapefile (`ne_110m_admin_0_countries.shp`)

#### Required Packages:
- tidyverse
- ggplot2
- plotly
- sf
- dplyr
- broom
- forcats
- htmlwidgets

#### Project Summary

This project explores FIFA 18 players using three main plots:

- Interactive visualization: Relationship between age and player rating. Interactive plots allows you to hover over points and see specifically what player that point is.
- Spatial analysis: Average player ratings by country
- Regression modeling: Effect of age and potential on overall rating
